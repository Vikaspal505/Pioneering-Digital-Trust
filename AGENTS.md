# CyberGuard Fusion — Agent Notes

This project is a TanStack Start (SSR + React) application implementing a
Real-Time Cyber-Fraud Fusion system with Private Federated Learning.

## Key conventions
- File-based routing lives in `src/routes/`. Root layout is `__root.tsx`.
- Do **not** create `src/pages/` or Next.js-style layouts.
- `routeTree.gen.ts` is auto-generated — do not edit by hand.
- All UI primitives are in `src/components/ui/` (Radix + Tailwind).
- Keep the `main` branch in a working, deployable state at all times.
